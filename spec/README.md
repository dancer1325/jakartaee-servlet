Jakarta Servlet Specification
=============================

* Jakarta Servlet Specification

Building
--------

* Prerequisites:
  * JDK8+
  * Maven 3.0.3+
* How to build?
  * `mvn install`
    * `target/generated-docs/servlet-spec-<version>.html` -- .html --
    * `target/generated-docs/servlet-spec-<version>.pdf` -- .pdf --
